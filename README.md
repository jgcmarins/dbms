# dbms
Database management system developed for college purpose.

This software is a file handler. It writes records dynamically.

It always writes in binary mode.



## how to use
### to compile:
```
make
```

### to run:
```
make run
```

### to run with valgrind:
```
make valgrind
```

### to clean written data:
```
make cleanfiles
```
